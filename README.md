# ADFMCOMMS2

[![Build Status](https://github.com/seisyuu-hantatsushi/ADFMCOMMS2.jl/actions/workflows/CI.yml/badge.svg?branch=main)](https://github.com/seisyuu-hantatsushi/ADFMCOMMS2.jl/actions/workflows/CI.yml?query=branch%3Amain)

## About
An adapter class for using SDR devices (AD-FMCOMMS2, ADALM-PLUTO) based on Analog Devices RF transceivers AD9361 and AD9363. 

## Examples
### FM Demodulator
